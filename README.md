Microwave
=========
Example of a microwave oven with working countdown timer,
control panel and light!

Implemented using pythons Tkinter library for the GUI and the State
design pattern.

Instructions
------------
* clone repository
* install tkinter (yum install tkinter)
* python microwave.py
